# Skills

Custom Claude Code skills (slash commands) used in the course. These go in your project's `.claude/skills/` folder. Each skill has its own subfolder with a `SKILL.md` file.

- `feature/` - Main workflow command for loading, starting, reviewing, testing and completing features
- `research/` - Generates documentation from research spec files into the `@/context/research` folder
- `cleanup/` - Runs housekeeping tasks like finding unused imports, stale TODOs and orphaned files
- `api-design/` - API design principles for Next.js App Router routes
- `brainstorming/` - Transforms vague ideas into validated designs through structured reasoning
- `code-reviewer/` - Deep code review for TypeScript/React/Next.js correctness and patterns
- `github-actions/` - GitHub Actions CI/CD specialist for Next.js pipelines
- `plan-writing/` - Structured task decomposition before implementation
- `playwright-expert/` - E2E testing patterns and Playwright MCP browser automation
- `test-master/` - Vitest unit and integration test specialist
